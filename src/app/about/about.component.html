À propos Tout savoir sur Daily Dashboard !

Daily Dashboard est un SaaS gratuit vous permettant de regrouper les différents services que vous consulter tous les jours comme vos agendas, la météos, les informations en un seul endroit.

L'application a été conçu avec les technos suivantes :

  • API en PHP avec Symfony
  • Client en Javascript avec Angular
  • API SNCF pour les horaires de trains.
  • API OpenWeatherMap pour la météo.
  • NewsApi.org pour les informations.
